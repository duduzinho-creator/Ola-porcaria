# Explicação do Projeto - Fundamentos 
## Do que se trata o projeto?
 Esse projeto será feito para explicar as disciplinas um pouco melhor para todos entenderem, ou seja, explicando matéria por matéria.
  
  Levando em consideração que somos todos calouros, precisamos de dicas para as matérias e com os professores também, esse projeto vai ajudar os calouros a se encontrarem melhor no curso.
