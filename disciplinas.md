# Programação I

Programação I é a disciplina onde você aprende os fundamentos da programação. É a base para praticamente todo o curso de Engenharia de Software.
#### Lógica de programação – como pensar passo a passo para resolver um problema.
#### Algoritmos – sequência de instruções para executar uma tarefa.
#### Variáveis – locais onde os dados são armazenados.
#### Tipos de dados – como números, textos, verdadeiro/falso.
#### Estruturas condicionais – decisões no programa (ex: se, senão).
#### Estruturas de repetição – repetir ações (ex: para, enquanto).
#### Entrada e saída de dados – receber informações do usuário e mostrar resultados

# Lógica de programação

Lógica de programação é a forma de organizar o pensamento para resolver problemas usando passos lógicos que depois podem ser transformados em um programa de computador.
Ela ensina a criar algoritmos, que são sequências de instruções para resolver um problema.
